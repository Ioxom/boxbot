# ioxbox
discord bot. boxes. yes

# running
### from a jar file:
1. go to [the upload-artifact section of the actions page](https://github.com/Ioxom/ioxbox/actions/workflows/upload-artifact.yml)
2. click on the most recent job
3. download and run

### from the sources
1. clone this repository
2. run `gradlew run`
